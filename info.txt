# AI Smart Study Planner & Learning Intelligence System - Documentation

## Project Overview
AI Smart Study Planner is a comprehensive intelligent learning management system designed for students to optimize their study schedules, track progress, and improve learning outcomes using AI and advanced analytics.

## Core Features
1.  **Smart Study Schedule Generator**: Automatically creates balanced study timetables based on subject difficulty, priority, and deadlines.
2.  **AI Chat Study Assistant**: An interactive bot that provides study tips, motivational support, and schedule adjustments based on student data.
3.  **Productivity Analytics Dashboard**: Visualizes study habits, completed topics, and productivity trends using interactive charts.
4.  **Smart Revision System (Spaced Repetition)**: Implements the 1-3-7-15 day rule to move knowledge from short-term to long-term memory.
5.  **Study Focus Mode (Pomodoro)**: A distraction-free timer with focus scoring to help students maintain high concentration during sessions.
6.  **Subject & Topic Management**: Allows students to organize their curriculum, set difficulty levels, and track completion at a granular level.
7.  **Predictive Analytics**: Calculates "Exam Readiness" and "Burnout Risk" based on historical study patterns.
8.  **Professional Navigation**: A responsive, modern Navbar integrated across all pages with active link highlighting and user profile management.

## Technical Stack
- **Frontend**: Next.js 15 (App Router), Tailwind CSS 4, Lucide Icons, Recharts, Framer Motion.
- **Backend**: Next.js API Routes, NextAuth.js (Authentication).
- **Database**: Prisma ORM with SQLite (dev.db).
- **AI Integration**: OpenAI/Gemini API (Simulated for Demo).
- **Date Handling**: date-fns.
- **Typography**: Plus Jakarta Sans (Headings), Inter (Body).
- **Branding**: "StudyIntel" with a Deep Indigo (#6366f1) and Soft Violet (#d946ef) theme.

## Project Structure
- `/src/app`: Next.js pages and API route entry points.
- `/src/components`: Reusable React components (e.g., `Navbar.tsx`, `AIQuiz.tsx`).
- `/backend`: Core backend system.
  - `/lib`: Business logic, services, and core utilities (SRS, Burnout logic, Gamification).
  - `/prisma`: Database schema and SQLite storage.
  - `/scripts`: System maintenance and verification tools.

## Installation & Setup
1.  **Install dependencies**: `npm install`
2.  **Set up environment variables** in `.env`:
    `DATABASE_URL="file:./dev.db"`
    `NEXTAUTH_SECRET="your-secret-key-here"`
3.  **Initialize database**: `npm run prisma:migrate`
4.  **Generate Prisma Client**: `npm run prisma:generate`
5.  **Run development server**: `npm run dev`
6.  **Build for production**: `npm run build`

## How to See/Manage Database (Visual Guide)
The project uses Prisma Studio, a powerful GUI to view and edit your data:
1.  Open your terminal in the project root.
2.  Run the command: `npm run prisma:studio`
3.  This will automatically open a browser window at `http://localhost:5555`.
4.  You can now:
    - View all tables (Users, Subjects, Topics, Achievements, etc.).
    - Manually add, edit, or delete records for testing.
    - Filter and search through your study data.

## How the Navbar Works
- **Component**: Located at `src/components/Navbar.tsx`.
- **Integration**: Included in the root `layout.tsx` to wrap all application pages.
- **Features**:
  - **Responsive Design**: Collapses into a hamburger menu on mobile devices.
  - **Dynamic Highlighting**: Uses `usePathname()` to highlight the current active page.
  - **Auth Integration**: Displays user name/email and provides a Logout button when signed in.
  - **Glassmorphism**: Uses Tailwind's `backdrop-blur` for a modern, professional look.
